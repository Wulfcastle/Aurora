Forward geocoding 

http://api.tiles.mapbox.com/v4/geocode/ {index} / {query} .json?access_token= <your access token> 
 
 Examples 
 http://api.tiles.mapbox.com/v4/geocode/mapbox.places/1600+pennsylvania+ave+nw.json

Reverse geocoding 

http://api.tiles.mapbox.com/v4/geocode/ {index} / {lon} , {lat} .json?access_token= <your access token> 
 
 Examples 
 http://api.tiles.mapbox.com/v4/geocode/mapbox.places/-73.989,40.733.json 
 http://api.tiles.mapbox.com/v4/geocode/mapbox.places/139.7786,35.6807.json

Batch requests 

http://api.tiles.mapbox.com/v4/geocode/ {index} / {query} ; {query} ; ... ; {query} .json?access_token= <your access token> 
http://api.tiles.mapbox.com/v4/geocode/ {index} / {lon} , {lat} ; {lon} , {lat} ; ... ; {lon} , {lat} .json?access_token= <your access token> 
 
 Examples 
 http://api.tiles.mapbox.com/v4/geocode/mapbox.places-permanent/20001;20009;22209.json
